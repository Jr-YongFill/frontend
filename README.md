<!-- GETTING STARTED -->
## Getting Started
1.  Clone the repo
```sh
   git clone https://github.com/Jr-YongFill/frontend
  ```
2. Build Project
  ```sh
   cd frontend
   npm i
  ```

4. Run
  ```sh
   npm start
  ```
